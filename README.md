# Plotly & Flask

## How to run Flask Server
1. Open terminal
2. Navigate to root of folder
3. Enter Command ``python app.py``

Next Open Your Browser and Visit the following url  [http://127.0.0.1:5000/plotly](http://127.0.0.1:5000/plotly)
